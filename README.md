# gradlew-update

Update your gradle wrapper to the latest and greatest

## Installation
```
npm install -g gradlew-update
```

## Usage
Within the directory of your Gradle project:
```
gradlew-update
```

## Dev
To test the command globally on your machine:
```
npm run link
gradlew-update
```
